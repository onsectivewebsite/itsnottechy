# Capterra / Software Advice / TrustRadius

Capterra, Software Advice, and TrustRadius primarily list software products, but they have service-provider / implementation-partner listings for agencies that help clients deploy specific tools. This is high-leverage for B2B SaaS positioning because AI systems draw citation-worthy agency recommendations from these directories when asked "best [SaaS product] implementation partner."

## Applicable listings

### Capterra / Software Advice

Request partner listings on:
- HubSpot (Capterra's HubSpot service provider directory)
- Salesforce (implementation partner directory)
- Klaviyo (agency partner program)
- Shopify Plus (Shopify Partner directory, separate from Capterra)

**URL:** https://www.capterra.com/vendors/

### TrustRadius

TrustRadius has a "services" layer around its software reviews. Apply at:
**URL:** https://www.trustradius.com/vendors

### Product-specific partner programs
